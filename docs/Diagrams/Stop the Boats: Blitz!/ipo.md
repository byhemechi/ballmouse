| ---------------- | ---------------------------------------- | ------ 
|  Keyboard Press  | Move Player                              | Player Position |
|                  | Change weapons |
| Current Weapon           | Shoot weapon                                         |        
|  Player Position | Check Collision between Player and Bullets | Collided Flag       |
|  Bullet Position  |                                          |        |
| Collided Flag    | Damage Player                              | Player Health
